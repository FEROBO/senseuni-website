# SenseUni Website — Project Context

## Project Type
Static HTML/CSS/JS website. No frameworks, no build tools, no WordPress.
Target audience: US/global B2B market. Deploy target: Cloudflare Pages.

## Brand Brief
See full brand strategy, IA, and page content in: docs/brand-guide.md

## Design System
- Primary bg: #1A1A1C (dark)
- Brand color: #2C2069
- Accent/CTA: #7435FD
- Highlight: #8B71FF
- Fonts: Sora (headings), Inter (body), Space Grotesk (data/labels)
- Style: Glassmorphism cards, deep dark background, purple gradient glow effects
- Reference: Stripe × Linear × Apple
- Animation level: Level 3 — every section moves, nothing distracts
  - Allowed: Scroll Reveal, Parallax, Gradient Flow, Number Count-Up, Card Hover, Background Glow
  - Avoid: Heavy 3D, particle overload, WebGL, anything that hurts performance

## Content Principle
- Headline: "Building Smarter Retail Networks."
- Lead with industry vision, not product features
- Avoid AI buzzwords and hollow slogans
- Emphasize: Deployment / Operations / Scalability / Business Impact

## Navigation (final, do not change)
Home | AI Smart Cabinet | Solutions | Technology | Case Studies | Company | Get a Demo

## Working Rules
1. One section at a time. Wait for screenshot confirmation before proceeding.
2. Never modify nav and footer once locked.
3. All copy must match brand-guide.md — do not invent new messaging.
4. CSS variables for all colors — never hardcode hex values inline.
5. Mobile-responsive by default.
